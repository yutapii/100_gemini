# 080_tps/ - TPS改善活動

## 概要

**TPS（Toyota Production System）**の考え方を取り入れた、継続的改善（Kaizen）のための活動記録フォルダ。

Gemini CLIの品質向上、問題解決、標準化を推進する。

## 改善活動の種類

### 1. 問題解決
- バグ・エラーの根本原因分析
- なぜなぜ分析（5 Whys）
- 再発防止策の立案

### 2. 標準化
- ベストプラクティスの文書化
- チェックリスト作成
- 手順書の整備

### 3. 効率化
- 作業時間の短縮
- 自動化の推進
- ツール改善
