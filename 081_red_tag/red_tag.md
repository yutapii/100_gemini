# 赤札リスト

## 赤札作戦とは

不要そうなファイルをいきなり削除せず、999_trashに移動して様子を見る。
30日経過して問題なければOSのゴミ箱へ移動（完全削除はしない）。

---

## ルール

1. 不要そうなファイル → `999_trash/` に移動
2. ファイル名に日付プレフィックス追加（例: `20251209_old_file.js`）
3. このファイルに記録（いつ・何を・なぜ）
4. **30日経過 → OSのゴミ箱へ移動**（`~/.Trash`）
5. やっぱり必要 → 元の場所に戻す（日付プレフィックス削除）

### 移動コマンド（macOS）
```bash
mv 999_trash/ファイル名 ~/.Trash/
```
