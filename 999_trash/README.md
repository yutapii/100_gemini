# 999_trash/ - アーカイブフォルダ

## ポリシー

**絶対ルール**: このフォルダ内のファイルは**削除禁止**

- 不要になったファイルは、削除せずにここに移動する
- 過去の知恵・試行錯誤の記録として保管
- 後で復元が必要になる場合に備える

## 使い方

### ファイルを移動する
```bash
# 例: 古いスクリプトを移動
mv old_script.py 999_trash/old_script_20260204.py
```

### 復元する
```bash
# 必要になったら元の場所に戻す
cp 999_trash/old_script_20260204.py ./old_script.py
```

## ファイル名規則

移動時は日付をつけることを推奨：
```
元のファイル名_YYYYMMDD.拡張子
```

例：
- `検査スクリプト.py` → `検査スクリプト_20260204.py`
- `config.json` → `config_20260204.json`

## 定期整理

**半年に1回**、本当に不要なファイルを確認：
- それでも削除せず、圧縮してアーカイブ
- `archive_YYYYMM.tar.gz` 形式で保存

## 禁止事項

❌ `rm -rf 999_trash/*` （一括削除禁止）
❌ 安易な削除
✅ 移動のみ
✅ 圧縮・アーカイブは可

---

**知恵のアーカイブ。削除は最後の手段。**
